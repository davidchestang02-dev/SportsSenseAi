# SportsSenseAi v1.0.0

Release date: 2026-04-08

This release marks the first launch-ready version of SportsSenseAi for MLB.

Included in this release:

- Cloudflare Workers backend with production MLB routes for auth, projections, simulation, market making, risk, auto-bet, billing status, and AI Q&A.
- Cloudflare D1 schema and migrations for launch data storage.
- Streamlit user dashboard in `web/` and admin console in `admin/`.
- Expo mobile app scaffold in `mobile/` wired to the production API.
- Automated test coverage across unit, integration, performance, regression, safety, and end-to-end smoke paths.
- GitHub Actions workflows for tests, Workers deploys, D1 migrations, and mobile/web deployment placeholders.

Launch state:

- Production API is live at `https://sportssenseai-api.david-chestang02.workers.dev`
- Billing is safely bypassed for launch and can be enabled later with Stripe credentials.
- AI Gateway-backed `/mlb/qa` is live in production.
