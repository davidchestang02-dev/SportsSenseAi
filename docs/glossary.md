# Glossary

`CompositeScore`
SportsSenseAi's blended decision score for ranking players.

`Run Environment`
Combined context score from weather, park, bullpen, and umpire effects.

`Market Maker`
The pricing layer that converts model probability into a posted line.

`Risk Engine`
The bankroll and exposure layer that sizes bets.

`AutoBet`
The execution-prep layer that turns approved edges into slips.
